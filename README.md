
# InfinityModelProjects

This repository is a collection of web development projects created as part of college MERN stack revision labs. The projects progress from basic HTML, CSS, and JavaScript to more advanced, interactive, and full-stack-ready applications.

## Repository Overview

- **Project1: Personal Profile**
  - A simple HTML profile page with sections for About, Skills, Education, and Contact.
  - Demonstrates semantic HTML and basic navigation.

- **Project2: Travel Destination – Manali**
  - A static site showcasing Manali as a travel destination.
  - Includes sections for About, Places to Visit (with images), Packages, and Contact.

- **Project3: Foodie’s Hub – Restaurant Portal**
  - Responsive restaurant portal using HTML, CSS, and JavaScript DOM manipulation.
  - Features:
    - Dynamic menu search and item highlight
    - Add-to-cart functionality with live order count
    - Rotating daily specials
    - Contact form with validation and feedback
  - See Project3/README.md for full details.

- **Project4: Book Haven**
  - Book showcase site with Bootstrap styling.
  - Displays book categories and cards with images and descriptions.

- **Project5: Drumkit App**
  - Interactive drumkit using HTML, CSS, and JavaScript.
  - Users can play drum sounds by clicking buttons or pressing number keys.

- **Project6: CourseHub Dashboard**
  - Online course information portal with a modern dashboard UI.
  - Features:
    - Add, search, and manage courses
    - Favorite marking and enrollment stats
    - Responsive design and dark mode toggle

## How to Use

1. **Clone or Download:**  
   Download this repository or clone it using:
   ```
   git clone <repo-url>
   ```
2. **Open Any Project:**  
   Navigate to a project folder and open index.html in your browser.
3. **Explore Features:**  
   Try out the interactive features in Project3, Project5, and Project6.

## Learning Goals

- Practice with semantic HTML and CSS layouts
- Apply JavaScript for DOM manipulation and interactivity
- Build responsive, accessible UIs
- Prepare for full-stack (MERN) development
